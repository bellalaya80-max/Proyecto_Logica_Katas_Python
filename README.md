# 🐍 Proyecto: Lógica y Katas de Python

### 📖 Descripción
Este proyecto consiste en una serie de 41 ejercicios prácticos (Katas) diseñados para fortalecer los cimientos de la lógica de programación en Python. El análisis recorre desde los fundamentos básicos (variables y condicionales) hasta conceptos avanzados de Programación Orientada a Objetos (POO) y funciones modulares. El objetivo principal es desarrollar la capacidad de resolver problemas técnicos y automatizar procesos, habilidades críticas para un Analista de Datos.
#
### 🗂️ Estructura del Proyecto
├── README.md            # Guía y descripción detallada del proyecto.
├── ejercicios.py        # Archivo principal con la resolución de los 41 retos.
#
### 🛠️ Instalación y Requisitos
Este proyecto ha sido desarrollado con Python 3.14. Para ejecutarlo, no se requieren librerías externas pesadas, ya que se centra en la lógica nativa del lenguaje.

Bibliotecas utilizadas: math (para cálculos geométricos de precisión).

Entorno recomendado: VS Code o cualquier terminal con Python instalado.

Para ejercutar el análisis: python ejercicios.py 
#
### 📊 Resultados y Conclusiones
**Modularidad y Escalabilidad:** Se logró implementar una función "maestra" (procesar_texto) que actúa como director de orquesta, demostrando cómo organizar código complejo de forma eficiente.

* **Robustez:** Se integró el manejo de excepciones (try-except) y la validación de tipos, asegurando que el programa no falle ante entradas incorrectas del usuario.

* **Normalización:** Se aplicaron técnicas de limpieza de texto (.strip(), .lower()) esenciales para el pre-procesamiento de datos.

* **Integridad Técnica:** Los ejercicios fueron desarrollados en colaboración con IA y apoyo externo (documentación y consultas técnicas) para asegurar una arquitectura profesional y entender estándares de código avanzados durante el proceso de aprendizaje, en especial los últimos 5.
